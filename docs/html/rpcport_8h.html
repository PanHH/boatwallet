Detailed Description

RPC wrapper header file internally used by RPC porting.

rpcport.h is header file to be included by RPC Porting such as curlport. Upper layer calling RPC wrapper functions should include rpcintf.h instead.
The exact implementation of the actual RPC mechanism is controlled by RPC_USE_XXX macros.
