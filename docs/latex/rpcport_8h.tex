\subsection{Detailed Description}
R\+PC wrapper header file internally used by R\+PC porting. 

\mbox{\hyperlink{rpcport_8h}{rpcport.\+h}} is header file to be included by R\+PC Porting such as curlport. Upper layer calling R\+PC wrapper functions should include \mbox{\hyperlink{rpcintf_8h}{rpcintf.\+h}} instead.~\newline
The exact implementation of the actual R\+PC mechanism is controlled by R\+P\+C\+\_\+\+U\+S\+E\+\_\+\+X\+XX macros.
